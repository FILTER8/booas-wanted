# BOOAS WANTED

A CC0 open-source poster maker for BOOA.

Create pixel-style “WANTED” posters from on-chain BOOA metadata with customizable typography, colors, and layout.

---

## 🔗 Links

- BOOA Project → https://www.khora.fun/booa  
- Creator → https://x.com/0xfilter8  

---

## 🧠 About

BOOAS WANTED is a visual tool to generate posters using BOOA metadata.

- Fully client-side
- Custom typography controls (size, spacing, shadow)
- Dynamic layout that adapts to content
- PNG export
- Pixel-perfect rendering

---

## ⚙️ Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- html-to-image

---

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

---

## 📦 Features

- Load BOOA by token ID
- Random BOOA generator
- Editable:
  - Title
  - Wanted text
  - Boundary text
- Color palette from metadata
- Export poster as PNG
